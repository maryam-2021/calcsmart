# 🧮 CalcSmart - Premium Multi-Calculator Website

CalcSmart is a high-performance, responsive calculator dashboard featuring 63+ specialized tools for finance, health, math, time, and unit conversions.

## 🚀 Getting Started

1.  **Rebuild the Site**: If you make any changes to `build.js` or `styles.css`, simply run:
    ```bash
    node build.js
    ```
2.  **View Locally**: Open `index.html` in any modern web browser.
3.  **Deployment**: This site is optimized for deployment on **Netlify** or **Vercel**.

## 🛠️ Features

*   **Financial Suite**: Retirement forecasters, ROI, Loan calculators, and more.
*   **Health Suite**: BMI, Ideal Weight, BMR, and Body Fat estimators.
*   **Math Suite**: Scientific calculators, Fraction solvers, and Prime Number checkers.
*   **Time Suite**: Time Zone converters, Business Day calculators, and Countsdowns.
*   **Unit Conversion**: High-precision length, weight, temperature, and data converters.

## 🎨 Design

*   **Glassmorphism Layout**: Modern, semi-transparent cards with a premium feel.
*   **Dark & Light Modes**: Full theme support with automatic system detection.
*   **Responsive**: Fully optimized for mobile, tablet, and desktop screens.
*   **Animations**: Micro-interactions and smooth transitions for a high-end experience.

## ⚙️ Maintenance

All HTML pages are procedurally generated from the `build.js` file. This makes it easy to add new calculators or update site-wide content (like headers or footers) in one single location.
